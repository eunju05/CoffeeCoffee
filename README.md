# :coffee: CoffeeCoffee

## 프로젝트 소개
+ 스테이지마다 주어진 시간 동안 음료를 조합해 손님에게 제공하는 게임입니다.
+ 스테이지는 총 7일로 이루어져 있으며 손님의 종류와 요구하는 음료의 종류가 늘어납니다.
+ 하트 점수가 0이 되면 게임 오버입니다.
+ 7일차가 끝난 뒤 최종 하트 점수에 따라 엔딩이 달라집니다.
+ 플레이 영상 : https://youtu.be/2IOzSWcGVN8
